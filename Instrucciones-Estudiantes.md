# Instrucciones para estudiantes

La guía actualizada de la práctica está en:

```text
docs/instrucciones-estudiante.md
```

Usa ese documento como referencia principal para:

- hacer fork o crear el repositorio desde template;
- clonar el repositorio;
- levantar la mini API Express con `npm install` y `npm start`;
- seguir las misiones de Git Flow;
- crear ramas, Pull Requests, tags y hotfixes;
- entender el cierre automático de issues por workflow.
